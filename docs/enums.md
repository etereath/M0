# 统一枚举

## 任务

- `TaskMode`：`SIMPLE`、`COMPLEX`、`CONTROLLED`
- `TaskStatus`：`DRAFT`、`PENDING_APPROVAL`、`READY`、`ASSIGNED`、`ACCEPTED`、`IN_PROGRESS`、`BLOCKED`、`SUBMITTED`、`NEEDS_REVIEW`、`COMPLETED`、`CANCELLED`
- `AssignmentStatus`：`PENDING`、`ACCEPTED`、`REJECTED`、`CANCELLED`、`COMPLETED`
- `AssignmentMethod`：`MANUAL`、`AGENT_SUGGESTED`、`AGENT_AUTO`、`REASSIGNED`
- `TaskUpdateType`：`ACCEPTED`、`REJECTED`、`STARTED`、`PROGRESS`、`BLOCKED`、`UNBLOCKED`、`SUBMITTED`、`COMPLETED`、`COMMENT`、`CORRECTION`、`EVIDENCE_ADDED`

## 项目

- `ProjectStatus`：`PROPOSED`、`PLANNING`、`ACTIVE`、`PAUSED`、`COMPLETED`、`CANCELLED`
- `MilestoneStatus`：`PLANNED`、`ACTIVE`、`BLOCKED`、`COMPLETED`、`CANCELLED`
- `ProjectUpdateType`：`PROGRESS`、`RISK`、`BLOCKER`、`NEXT_ACTION`、`STATUS_SUGGESTION`、`COMMENT`、`CORRECTION`

## 人员和权限

- `EmployeeRole`：`OWNER`、`MANAGER`、`TEAM_LEADER`、`WORKER`、`OBSERVER`
- `ActorType`：`HUMAN`、`AGENT`、`SERVICE`

## 风险、审批和证据

- `RiskLevel`：`LOW`、`MEDIUM`、`HIGH`、`CRITICAL`
- `ApprovalStatus`：`PENDING`、`APPROVED`、`REJECTED`、`CANCELLED`、`EXPIRED`
- `EvidenceType`：`PHOTO`、`VIDEO`、`TEXT`、`FILE`、`MEASUREMENT`、`MANAGER_CONFIRMATION`

## 库存

- `InventoryTransactionType`：`PURCHASE_IN`、`RETURN_IN`、`USE_OUT`、`ADJUST_IN`、`ADJUST_OUT`、`TRANSFER`、`DAMAGE_OUT`、`EXPIRED_OUT`、`RESERVE`、`RELEASE`

所有数量字段均传正数，由流水类型决定对 `on_hand`、`reserved` 和 `available` 的影响。

## 同步

- `SyncDirection`：`CLOUD_TO_LOCAL`、`LOCAL_TO_CLOUD`
- `SyncOperation`：`CREATE`、`UPDATE`、`APPEND`、`SOFT_DELETE`、`RECONCILE`
- `SyncStatus`：`PENDING`、`PROCESSING`、`SUCCEEDED`、`FAILED_RETRYABLE`、`FAILED_PERMANENT`、`CONFLICT`

## 审计

- `AuditResult`：`SUCCEEDED`、`DENIED`、`FAILED`、`NO_OP`
